# Robot HTML tĩnh

Repo này chứa các trang HTML **chạy độc lập**, không cần server, không API.

## 1. Robot theo ngày
- Tự chọn file theo ngày thực tế (âm lịch, lễ, mặc định).
- Cách chạy:  
  `index.html` → gọi `robot.html` → chọn file phù hợp → hiển thị.

Mở tại:  
https://namnguyen7133-png.github.io/HTML-i-n-tho-i-/

## 2. Robot outfit (trang phục)
- Gợi ý trang phục theo nhiệt độ / dịp.
- Các file nằm trong thư mục `web/`.
- Cách dùng:
  - Sửa dữ liệu trong `outfits-data.js`
  - Mở `generator.html`
  - Sinh nội dung → dán vào các file `300.html` đến `399.html`
